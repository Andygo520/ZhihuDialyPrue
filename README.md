##制作日程

###1.制作启动界面 2015-5-22

1）通过设备尺寸构建API请求

2）使用HttpURLConnection下载网络数据

3）使用文件制作图片作者和图片缓存

###2.使用View Animation制作启动页渐隐动画 2015-5-23

Error )动画有时间，监听动画播放完毕之后再跳转到下一个MainActivity，
